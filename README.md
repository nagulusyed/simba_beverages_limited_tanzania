# Simba Beverages Limited – Tanzania

Welcome to the premium landing page for **Simba Beverages Limited (Tanzania)**. This project serves as a high-end wholesale and B2B portal for our signature beverage line, featuring our flagship product, **Vin Nkolomoka**.

## Features

- **Premium Aesthetic**: A dark, luxurious theme accented with gold elements, glassmorphism, and sleek micro-animations to reflect the premium quality of the beverages.
- **Interactive Pricing Calculator**: A dynamic wholesale calculator that instantly applies volume discounts, calculates total savings, and updates dynamically as users drag the slider.
- **Seamless Enquiry Flow**: Clicking "Create Enquiry With Quote" smoothly scrolls the user to the contact form and automatically pre-fills the message field with a detailed summary of their potential order.
- **Dual Send Options**: Prospects can submit their enquiries via traditional Email (Formspree) or instantly via WhatsApp.
- **Interactive Product Range**: A beautiful grid showcasing the product lineup (Vin Nkolomoka, Nkolo Mboka, Ola Kombucha, Ginger Punch, and Hard Rock Energy).
- **Premium Instagram Integration**: A stylish social media section featuring animated gold text, custom hover states for image placeholders, and a premium gradient button linking to the official Instagram page.

## Technologies Used

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Custom properties (variables), CSS Grid/Flexbox, transitions, keyframe animations, and custom scrollbars for a premium feel. No heavy frameworks required.
- **Vanilla JavaScript**: DOM manipulation, interactive pricing logic, custom dropdown implementations, and smooth scrolling.

## Setup & Deployment

Since this is a static website, you can simply open `index.html` in any modern web browser to view the project locally.

To deploy, upload the files to any static hosting provider (e.g., GitHub Pages, Vercel, Netlify, or standard cPanel hosting). Ensure that you update the `FORMSPREE_ID` inside `index.html` with your actual Formspree endpoint if you wish to capture email submissions.
